OsmAnd 3.9 (Android)

December 07, 2020

We are glad to announce the release of version 3.9! We proudly present some major improvements and new features, and hope you will enjoy them:

- Added option to export and import all data including settings, resources, my places - Plan Route: added graphs for segments with the route, added the ability to create and edit multiply segment tracks – Added OAuth authentication method for OpenStreetMap, improved UI of OSM dialogs – Custom color for favorites and tracks waypoints
  • Option export/import
  • Improved "Plan a route" tool
  • Improved OSM editing Plugin
  • Added custom color for favorites and track waypoints
  • What else is in this release?

    • Option export/import

    We added option to export and import all data including settings, resources, my places. It's a big step to backup feature, which we will plan to add the next release.

    Improved "Plan a route" tool

    Our Android team added graphs for segments with the route, added the ability to create and edit multiply segment tracks in "Plan a route" feature.

    Improved OSM editing Plugin

    Now, you can add custom color for favorites and track waypoints.

    What else is in this release?

    Introduced subscription on-hold/pause.

    Added large "Dismiss" button for Navigation.

    Fixed description pole of OSM editing.

    Fixed different distance: Plan Route vs GPX Overview.

    Fixed crash when taping on A/V notes widget.
