OsmAnd 3.90 (iOS)

December 15, 2020

Hi!

As we said, we are working on improving the ios version of the application. The proof is this release. We're glad to announce the new OsmAnd version for iOS.

Thank you for your support and feedback.

Furthermore in the article, there is a final list of extensions and fixes:

  • Introduced “Plan a route" tool
  • Added import all date types
  • Added POI text labels
  • What else is in this release?

    • Introduced “Plan a route" tool

    At this release we added new tool named "Plan a route": menu -> Plan a route. Now, you can create your trip by using one or more navigation profiles, saved it like GPX-file.

    Added import all date types

    When you import obf-file (Application profile) you can select data to be imported: renderting styles, quick actions, routing, trips, maps, favorites and etc.

    Added POI text labels

    For switching on POI text labels go to menu -> Map -> POI overlay labels.

    What else is in this release?

    • Returned Wikipedia to POI Overlay menu and Search Categories

    • Fixed bugs

    And remember that only together we can achieve the best results!

    New features are coming SOON!
