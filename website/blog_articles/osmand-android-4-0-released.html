OsmAnd 4.0 (Android)

July 02, 2021

Great news for Android users!

We're happy to announce the new 4.0 OsmAnd release for Android devices.

At this release we changed our monetization and added new functions.

Update your OsmAnd, check out all the new features and let us know if you like everything!

  • New monetization (Google Play version)
  • Cloud backup and restore
  • Moved 'OsmAnd live' updates menu
  • Added night mode for Topo rendering style
  • Added option to download Contour lines in feet
  • Updated 'Trip recording' dialog
  • Tracks can now be colored by altitude, speed, or slope
  • Added option to change the route line appearance
  • 'Distance by tap' moved from 'Radius ruler' tool into a separate option
  • 'Plan Route' improvements
  • New Purchases screen
  • New POI types
  • New Language dialogue for Voice prompts
  • My places: last opened tab is now remembered and will be open next time
  • What else is in this release?

    • New monetization (Goolge Play version).

    At this release we made two purchase plans for OsmAnd (Google Play):

    • 'Maps+' plan has one-time payment and annual subscription. It's light paid version with all needed features: monthly map updates, unlimited map downloads, offline Wikipedia and Wikivoyage, terrain maps, nautical depth.

    • 'OsmAnd Pro' plan has monthly and annual subscriptions. It's a full paid version with all paid features: OsmAnd Cloud, advanced widgets, hourly map updates, monthly map updates, unlimited map downloads, offline Wikipedia and Wikivoyage, terrain maps, nautical depth.

    Full information you can find here.

    Cloud backup and restore.

    New paid function for 'OsmAnd Pro' subscription.

    This function allows to make backup and restore all your OsmAnd data: settings, my places, resources. Size of storage is 3Gb.

    Go to Menu -> Settings -> Backup & Restore. Select the data to be exported to backup storage.

    Moved 'OsmAnd live' updates menu.

    'OsmAnd Live' updates (function of hourly map updates) moved to "Menu -> Downloads maps -> Updates".

    Added night mode for Topo rendering style

    We added night mode for Topo and Offroad rendering styles.

    Added option to download Contour lines in feet

    Now, there is ippotunity to download Contour lines data in feet.

    Updated "Trip recording" dialog

    We improved and updated "Trip recording" dialog.

    Now you can see more detailed information and graph of your trip.

    You can set needed parameters for recording and start record of your trip by one tap.

    Tracks can now be colored by altitude, speed, or slope

    Now, you track can by colores by altutude, speed, or slope.

    Choose your track on the map or in 'My places -> Tracks', tap to 'Appearance' button.

    Added option to change the route line appearance

    User can choose routeline appearance for route line:

    Menu -> Configure profile (choose app profile) -> Profile appearance -> Customize route line

    'Distance by tap' moved from 'Radius ruler' into a separate option

    We separted 'Distance by tap' option from 'Radius ruler' tool.

    Now, you can find it in Menu -> Configure screen -> Distace by tap.

    'Plan Route' improvements

    At this release we continued to improve our 'Plan Route' tool.

    Now, you can see a landscape of your trip planning by switching between points and graphs tabs.

    Next, we added ability to change navigation options, accessing configure map and search without leaving 'Plan route' tool.

    New Purchases screen

    Now, ther is new 'Purchases' screen with detailed information about purchasing and Restore purchases button.

    New POI types

    Added new POI types, clubs: music, social, scout, freemasonry.

    New Language dialogue for Voice prompts

    Redesigned dialog for selecting the language for navigation instructions. Now it's more ease to choose and download needed language and type of voice guidance.

    My places: last opened tab is now remembered and will be open next time

    Your last opened tab in 'My places' menu is remembered. And this tab will be open next time.
