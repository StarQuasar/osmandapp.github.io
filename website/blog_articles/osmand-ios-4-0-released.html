OsmAnd 4.0 (iOS)

June 04, 2021

Hi, Everybody!

Great news for iOS OsmAnd users!

We're glad to announce the new OsmAnd version for iOS.

Update your OsmAnd, check out all the new features and let us know if you like everything!

We promise to continue improving the OsmAnd app and make it even more useful for trips and everyday rides to make your life set.

Furthermore in the article, there is a final list of extensions and fixes:

  • New "Add favorite" screen
  • Coordinates widget
  • Custom plugin
  • Extended the import/export functionality
  • Distance by tap moved from the Radius ruler
  • Improve CarPlay functionality and fixed bugs
  • Plugins menu
  • Added support for custom routing and rendering files
  • What else is in this release?

    • New "Add favorite" screen

    New "Add favorite" screen with options to select the custom icon, color, and shape.

    Now you can add your Favorites and Waypoints with custom style.

    Coordinates widget

    Added new "Coordinates widget" with coordinates of the current location:

    Menu -> Configure screen -> Coordinates widget

    Custom plugin

    Added ability to import custom user-defined plugins. Now, each user can create own plugin for OsmAnd.

    Menu -> Plugins -> Custom plugins

    Extended the import/export functionality

    Significantly extended the import/export functionality of Application profile. Complete data import / export compatible with other devices.

    'Distance by tap' tool moved from the Radius ruler

    'Distance by tap' tool moved from the Radius ruler into a separate option:

    Menu -> Configure screen -> Distance by tap

    Improve CarPlay functionality and fixed bugs

    Fixed CarPlay app freezes when iPhone screen is turned off.

    Choosing app profile that will be used while connected to CarPlay:

    Menu -> Settings -> OsmAnd Settings -> CarPlay profile

    Plugins menu

    Plugins screen moved from the 'Maps & Resources' to the main menu:

    Menu -> Plugins

    Added support for custom routing and rendering files

    Added support for custom routing and rendering files.

    What else is in this release?

    • Fixed UI in landscape orientation for “Plan a route” screen.

    • Fixed performance of Map markers direction line.

    • Fixed issue with importing GPX files with uppercase extensions.

    • Fixed issue with "Follow track" menu.

    • Fixed import of OBF-files.

    And remember that only together we can achieve the best results!

    New features are coming SOON!
